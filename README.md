# Tampermonkey-scripts

## ChatGPT Read Aloud Button Restore

Restores the "Read aloud" button to its original position right under every ChatGPT message

*Important:*
 - *It is required to enable "Allow User Scripts" option in Tampermonkey settings*
 - *The "Try again" and "More actions" buttons are permanently hidden now*
 - *It is recommended to exclude chatGPT url from uBlock extension, as it may conflict with the script and produce multiple errors in the browser console*

[One-click Install via Tampermonkey](https://www.tampermonkey.net/script_installation.php#url=https://raw.githubusercontent.com/iLintu/Tampermonkey-scripts/refs/heads/master/chatgpt-read-aloud-button-restore.user.js)
